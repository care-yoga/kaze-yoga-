# 風ヨガ ウェブサイト デザイン ブレインストーミング

## 選定デザイン：**禅的ミニマリズム × 有機的曲線**

### デザイン哲学
**呼吸と静寂を視覚化する**—ヨガの本質である「呼吸」「感覚への気づき」「心身の調和」を、デザインの余白・曲線・色彩で表現する。派手さを排除し、見る人に「落ち着き」「安心感」「やさしさ」を与える。

---

## 採用デザイン詳細

### 1. デザイン運動
**禅的ミニマリズム + 有機的曲線**
- 参考テンプレートの波のモチーフを活用
- 日本の伝統美学（余白、調和、自然）を現代的に解釈
- 過度な装飾を避け、必要な要素のみを配置

### 2. コア原則
1. **余白の力**：空間を「何もない」のではなく「呼吸する空間」として設計
2. **有機的曲線**：直線ではなく、波・呼吸のような柔らかい曲線を使用
3. **限定的色彩**：3色以下に絞り、落ち着きと一体感を表現
4. **段階的な情報開示**：スクロールに応じて背景色が変化し、ユーザーの「気づき」を促す

### 3. 色彩哲学
| 色 | 色コード | 用途 | 意図 |
|---|---|---|---|
| **深緑** | `#4a6c4c` | アクセント、見出し | 自然・安定・信頼・呼吸 |
| **白** | `#ffffff` | 背景（初期） | 清潔・開放感・呼吸の始まり |
| **ベージュ** | `#f4f1ea` | 背景（スクロール後） | 温かさ・落ち着き・呼吸の深まり |
| **濃いグレー** | `#2c3e50` | 本文 | 読みやすさ・落ち着き |

**色彩の遷移**：ページ上部は「白」で始まり、スクロールに応じて「ベージュ」へ変化。これは呼吸の「吸う」から「吐く」への流れを表現。

### 4. レイアウトパラダイム
- **ヒーローセクション**：90vh の大きな画像 + 波のアニメーション + テキストオーバーレイ
- **セクション区切り**：波のSVGアニメーションで自然な区切り
- **コンテンツ配置**：中央揃え + 最大幅1000px で落ち着きを演出
- **グリッドレイアウト**：ページ情報は3列グリッド（インストラクター、スケジュール、参加費など）

### 5. シグネチャー要素
1. **波のアニメーション**：セクション間の境界線として、複数層の波が動く
   - 4層の波（透明度が異なる）
   - スクロール時に色が白→ベージュに変化
   - 無限ループで「呼吸」を表現

2. **円形の統計情報**：参加者数、開催回数などを円形で表示
   - 深緑の枠線で統一感を出す
   - 中央に数字、下部にラベル

3. **柔らかいシャドウ**：`box-shadow: 0 4px 15px rgba(0,0,0,0.05)` で奥行きを表現

### 6. インタラクション哲学
- **スクロール連動**：背景色の段階的変化で、ユーザーの「気づき」を促す
- **ホバーエフェクト**：画像が1.1倍にスケールして、ユーザーの注意を引く
- **スムーズな遷移**：すべてのアニメーションに `transition: 1s ease` を使用

### 7. アニメーション
- **波のアニメーション**：`animation: move-forever 25s cubic-bezier(.55,.5,.45,.5) infinite`
  - 各層で異なる速度（7s, 10s, 13s, 20s）
  - 遅延（-2s, -3s, -4s, -5s）で重なり感を表現
  
- **フェードイン**：`.proj-item` は初期状態で `opacity: 0; transform: translateY(30px)` 
  - スクロール時に `.visible` クラスが付与され、フェードイン
  
- **ホバーアニメーション**：`.proj-item:hover .p-img { transform: scale(1.1); }`

### 8. タイポグラフィシステム
| 要素 | フォント | サイズ | 用途 |
|---|---|---|---|
| **見出し** | Tenor Sans | 3rem / 2rem | 大きな存在感、エレガント |
| **本文** | Zen Old Mincho | 1.2rem / 1rem | 読みやすさ、日本的な美学 |
| **キャプション** | Zen Old Mincho | 0.8rem | 補足情報 |

**フォント選択理由**：
- **Tenor Sans**：モダンで落ち着きがある、ヨガの「今この瞬間」を表現
- **Zen Old Mincho**：日本の伝統を感じさせ、ヨガの東洋的な背景を表現

## 実装ガイドライン

### Tailwind CSS での再現
- 色変数：`--green: #4a6c4c`, `--beige: #f4f1ea`, `--text: #2c3e50`
- 波のアニメーション：カスタムCSS（Tailwind では難しいため、`index.css` に記述）
- スクロール検知：JavaScript で `body` に `.scrolled` クラスを付与

### コンポーネント構成
1. **Header/Navigation**：シンプルなナビゲーション（ロゴ + メニュー）
2. **HeroSection**：背景画像 + 波アニメーション + テキストオーバーレイ
3. **WavesDivider**：セクション間の波のアニメーション
4. **SectionCard**：各セクション（概要、インストラクター、スケジュール等）
5. **Footer**：深緑背景 + 白テキスト

---

## デザイン決定の根拠

このデザインを選んだ理由：
1. **ヨガの本質を視覚化**：波のアニメーションで「呼吸」を表現
2. **参考テンプレートとの親和性**：BIOテンプレートの波・色彩・余白を活用
3. **ユーザーの「気づき」を促す**：スクロール時の背景色変化で、ユーザーの感覚を刺激
4. **シンプルで拡張性が高い**：後から画像・テキストを差し替えやすい設計
5. **モバイル対応**：余白と曲線を活かしたレスポンシブデザイン
